# Project Name: Final Calculator Project

## Project Description
This project is a console-based Java calculator application developed across multiple phases of the course and combined into one final program. The application performs basic mathematical operations, stores one numeric value in memory, manages a collection of integers, and uses exception handling to prevent crashes from invalid input.

## Project Features
- Add, subtract, multiply, and divide operations
- Single-value memory functions
- Integer collection memory for up to 10 values
- Display stored values
- Remove values by index
- Count stored integers
- Calculate sum, average, and difference between first and last values
- Exception handling for invalid numeric input
- Division-by-zero protection
- Menu-driven console interface

## Project Skills Learned
- Java syntax and program structure
- Classes and methods
- Arrays and data collections
- Exception handling
- User input validation
- Debugging and testing
- GitHub project management

## Language Used
- Java

## Development Process Used
This project was developed over several weeks in phases. Each phase added a new part of the calculator, including user interaction planning, design structure, memory features, and exception handling. The final version combines all those parts into one complete application.
